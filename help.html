ГРИС "Eye"

    Описание команд:
  1. Шаг - исполнитель делает смещается на 1 клетку, оставляя за собой след
  2. Прыжок - исполнитель делает смещается на 1 клетку, не оставляя за собой следа
  3. Поворот - исполнитель поворачивается против часовой стрелки
  4. Ветвление - исполнитель выполняет определенный блок в зависимости от условия
  5. Цикл - исполнитель выполняет определенный блок пока условие истинно
  6. Подпрогрмма:
    • Создать - создает новую подпрограмму с пустым телом (body)
    • Вызвать - вызывает подпрограмму в тело алгоритма
    Управление:
  • Для выделения команды - щелкните по ней левой клавишей мыши
  • Для перемещения команды вверх по списку нажмите клавишу "вверх" на клавиатуре
  • Для перемещения команды вниз по списку нажмите клавишу "вниз" на клавиатуре
  • Для удаления команды из списка нажмите клавишу "delete" на клавиатуре
