Описание

Графический исполнитель (ГРИС) - это программа, которая выполняет команды, заданные учеником. В нашем случае это Глаз (Eye), с которым мы и будем работать все время.

Меню в левой части программы позволяет нам задавать алгоритм, которому будет следовать Глаз. Если во время прохождения алгоритма Глаз наткнется на стену, то будет выведена ошибка (пункт алгоритма засветится красным).

Алгоритм

За создание алгоритма отвечает меню в левой части программы. Значение команд аналогично другим ГРИСам (Шаг - смещение на 1 клетку, оставляя за собой след; Прыжок - шаг на 1 клетку не оставляя за собой следа; Поворот - поворот против часовой стрелки на 90 градусов)

Данный ГРИС поддерживает ветвление и цикл (осбенностью является выполнение определенного количества повторов цикла). Также есть возможность использовать подпрограммы (процедуры).

Управление

Управление осуществляется после выбора пункта алгоритма (для этого щелкните левой клавишей мыши по пункту алгоритма, который распологается чуть ниже меню).

Для перемещения (пункта) вверх/вниз по списку нажмите клавишу "вверх"/"вниз" на клавиатуре соответственно; Для удаления пункта алгоритма нажмите клавишу "delete" на клавиатуре

Стены

ГРИС Eye поддерживает т.н. "стены" - это препятствия, пройти через которые Глаз не может. Для их расстановки (или удаления) просто щелкните по нужной клетке поля.
